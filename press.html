Update Beta v1.1.16

Published in Changelog

Hello Beta Testers!

Thank you all for testing Oueve! After a month of internal testing for notifications, we've rewritten notifications to include new features and improvements that we'll hope that you'll like.

We've also escalated the version number of Oueve to 1.1 to mark that all elements of the app have been redesigned or rewritten. Starting with beta v0.12, we aimed to improve all of the original code and graphics and inch closer to release. After several beta releases, v0.13 introduced UI improvements for the Data and Home Screen to help visibility and provide more useful data to the user. In beta v0.16, we introduced redesigns for the how notifications worked and improved the whole system with Snooze, Repeat, Auto-Add, and Quick-Add options and increased the version to v1.1. We're very close to release, but we still need your help so keep sending us feedback!


Looking forward to Winter Break,
The Oueve Team

Message About Updates

Published by Derek Lee

Hey!

We know we may have let you down recently. Initially, we planned to release Oueve with basic features through beta testing and slowly expand the feature base. However as we added more features, we found it hard to release a early release candidate of Oueve through beta testing channels. So over the last two years behind the scenes, we added more technical features allowing our user to keep track of their water consumption. We also spent the last two years improving the science behind Oueve, increasing the accuracy of the recommendations that our app could provide.

The trust that we intially built, connecting over 300 users, has been tested, but we are certain that we can finally release Oueve. In the future, we plan to continually update Oueve with regular software updates that include new and easier ways to track your water consumption.


Project Leader,
Derek Lee
